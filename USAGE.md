# 打车发票 & 加班记录核验系统 - 使用说明

## 功能简介

本系统可以自动识别打车发票信息，并与加班记录进行日期匹配，生成：
1. **交通费用明细汇总表** - 可编辑、导出图片/CSV
2. **加班凭证** - 标注了匹配日期的加班记录图片
3. **OCR 复核面板** - 查看原始识别结果，方便人工核对

## 快速启动

### 1. 配置 API Key

编辑 `.env` 文件，填入腾讯混元 API Key：

```env
HUNYUAN_API_KEY=你的API_Key
PORT=3000
```

> 获取 API Key：https://console.cloud.tencent.com/hunyuan/api-key

### 2. 安装依赖（首次运行）

```bash
npm install
```

### 3. 启动服务器

```bash
npm start
```

服务器启动后，访问：http://localhost:3000

## 使用流程

1. **上传发票** - 点击左侧卡片，选择多张打车发票图片
2. **上传加班记录** - 点击右侧卡片，选择当月加班记录截图
3. **开始分析** - 点击「🔍 开始分析」按钮
4. **查看结果**：
   - 交通费用明细汇总表（可编辑部门/工号/姓名）
   - 加班凭证（红框标注匹配日期）
   - OCR 复核面板（查看原始识别结果）
5. **导出**：
   - 汇总表 → 导出图片 / 导出 CSV
   - 加班凭证 → 导出图片

## 文件结构

```
├── server.js          # 后端服务器（提供 OCR API）
├── public/
│   └── index.html     # 前端页面
├── .env               # API Key 配置（需自行填写）
└── package.json       # 项目依赖
```

## 注意事项

- API Key 仅存储在服务器端，不会暴露给前端
- 图片在内存中处理，不会保存到磁盘
- 建议单次上传发票不超过 20 张（避免 API 限流）
