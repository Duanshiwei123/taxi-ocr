# 打车发票 & 加班记录核验系统

## 快速启动

### 1. 安装依赖
```bash
npm install
```

### 2. 配置 API Key
```bash
# 复制示例文件
copy .env.example .env

# 用记事本打开 .env，把 your_api_key_here 替换为真实的混元 API Key
notepad .env
```

### 3. 启动服务
```bash
npm start
```

启动后终端会显示：
```
✅ 服务已启动：http://localhost:3000
```

### 4. 访问使用
- **本机使用**：浏览器打开 http://localhost:3000
- **局域网其他人使用**：把 `localhost` 换成你的本机 IP，例如 http://192.168.1.100:3000
  - 查本机 IP：命令行执行 `ipconfig`，找 IPv4 地址

---

## 功能说明
1. 上传多张打车发票图片（实拍/电子发票均可）
2. 上传当月加班记录截图
3. 点击「开始分析」，系统自动：
   - OCR 识别每张发票的**实际乘车日期和乘车时间段**（非申请/审核时间）
   - OCR 识别加班记录中的所有加班日期
   - 用乘车日期匹配加班日期
4. 输出：
   - **交通费用明细汇总表**（可编辑部门/工号/姓名，支持导出图片/CSV）
   - **加班凭证**（加班记录图片上用红框标注匹配行 + 汇总凭证表）
   - **OCR 原始结果复核面板**（方便人工排查未匹配情况）
